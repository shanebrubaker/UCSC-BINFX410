# Data Directory

## Overview

This directory contains synthetic genomics data for the Spark exercise module.
Data is generated by `generate_data.py` and is **not** checked into version control.

## Generating Data

```bash
# Generate small dataset (1,000 variants) - for quick testing
python generate_data.py --size small

# Generate medium dataset (100,000 variants) - main exercise dataset
python generate_data.py --size medium

# Generate large dataset (1,000,000 variants) - performance comparisons
python generate_data.py --size large
```

All generation is **deterministic** (seeded with numpy seed=42), so every run
produces identical output.

## File Descriptions

### variants_{size}.csv

Main variant call data in CSV format with the following schema:

| Column       | Type   | Description                                         |
|--------------|--------|-----------------------------------------------------|
| CHROM        | string | Chromosome (chr1-chr22, chrX, chrY)                 |
| POS          | int    | Genomic position on chromosome                      |
| ID           | string | Variant ID (rs-number or `.` for novel)             |
| REF          | string | Reference allele (A, C, G, T)                       |
| ALT          | string | Alternate allele (A, C, G, T, differs from REF)     |
| QUAL         | float  | Variant quality score (0-100, Phred-scaled)         |
| FILTER       | string | Filter status (PASS, LowQual, LowDP, LowGQ)        |
| DP           | int    | Read depth (1-100)                                  |
| AF           | float  | Allele frequency (0-1)                              |
| GQ           | int    | Genotype quality (0-99)                             |
| GENE         | string | Associated gene name (or `.` for intergenic)        |
| CONSEQUENCE  | string | Variant consequence type                            |

**Consequence types**: `missense_variant`, `synonymous_variant`, `intron_variant`,
`intergenic_variant`, `stop_gained`, `splice_donor_variant`, `frameshift_variant`

**Data characteristics**:
- Chromosome distribution matches real human genome proportions (GRCh38)
- Quality scores are bimodal (80% high quality ~85, 20% low quality ~15)
- DP and GQ are positively correlated (as in real data)
- Allele frequencies follow a beta distribution skewed toward rare variants

### genes.bed

Gene annotations in BED format (tab-separated, no header):

| Column    | Type   | Description          |
|-----------|--------|----------------------|
| chrom     | string | Chromosome           |
| start     | int    | Gene start position  |
| end       | int    | Gene end position    |
| gene_name | string | Gene symbol          |
| strand    | string | Strand (+ or -)      |

Contains 500 gene records using real human gene names (BRCA1, TP53, EGFR, etc.).

### samples.csv

Sample metadata (CSV with header):

| Column     | Type   | Description                          |
|------------|--------|--------------------------------------|
| sample_id  | string | Unique sample identifier             |
| population | string | Population (EUR, AFR, EAS, SAS, AMR) |
| sex        | string | Sex (male, female)                   |
| phenotype  | string | Phenotype (case, control)            |

Contains 200 sample records.

### known_variants.csv

Known pathogenic variants for join exercises (CSV with header):

| Column                | Type   | Description                                  |
|-----------------------|--------|----------------------------------------------|
| CHROM                 | string | Chromosome                                   |
| POS                   | int    | Genomic position                             |
| ID                    | string | Variant ID (rs-number)                       |
| clinical_significance | string | Clinical classification                      |
| disease               | string | Associated disease                           |

Contains 1,000 records with clinical significance levels:
`pathogenic`, `likely_pathogenic`, `benign`, `likely_benign`, `uncertain_significance`
