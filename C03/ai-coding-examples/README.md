## How to Use These Examples

1. **Open the notebook** in Jupyter Lab, VS Code, or your preferred environment
2. **Read the explanations** in the markdown cells
3. **Run the code cells** to see examples in action
4. **Try the prompts** with Claude Code in your terminal
5. **Complete the practice exercises** at the end of each notebook

## Getting Started

```bash
# Navigate to the examples directory
cd ai-coding-examples

# Start Jupyter
jupyter lab
# or
jupyter notebook

# Or open in VS Code with the Jupyter extension
code .
```

## Each Notebook Contains

- **Learning Objective** - What you'll learn
- **The Prompt** - Example prompts to try with Claude Code
- **Working Code** - Runnable examples you can experiment with
- **Explanations** - Key concepts and best practices
- **Practice Exercises** - Hands-on challenges to reinforce learning

## Tips for Students

1. **Don't just read** - Run the code cells and experiment!
2. **Try variations** - Modify the code and see what happens
3. **Use Claude Code** - Practice the prompts in your terminal
4. **Build on examples** - Use these as starting points for your own projects

## Requirements

- Python 3.10+
- Jupyter Lab or Notebook
- pytest (for testing examples)
